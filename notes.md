# 1. User Interface:
### The admin side will have an interface where admins can view and manage certificate requests.

# 2.Details Submission:
### Admins will enter the Name, Course and Date of Certificate approval.

# 3.Certificate Generation:
### After Entering details, Certificate PDF should be generated automatically for provided details. 
### The generated certificate PDF will be saved to google drive and its link will be saved in DB.

# 4.Certificate Link Storage:
### The link to the generated certificate PDF will be stored in a database along with the student's email.
